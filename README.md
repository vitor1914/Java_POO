# Projeto Java Orientado a Objetos

Este é um projeto Java orientado a objetos que demonstra conceitos de programação orientada a objetos. O projeto inclui classes e exemplos que ilustram o uso de objetos, herança, polimorfismo, encapsulamento e outros princípios importantes de OOP.

## Requisitos

- Java Development Kit (JDK) instalado
- IDE de sua escolha (Eclipse, IntelliJ, etc.)

## Como usar

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/projeto-java-oop.git
   ```

2. Abra o projeto em sua IDE.

3. Explore as classes e pacotes para ver exemplos de programação orientada a objetos em Java.

4. Execute o projeto para ver os resultados dos exemplos.

## Exemplos

- `ExemploObjeto.java`: Demonstração simples de criação de objetos em Java.
- `ExemploHeranca.java`: Exemplo de herança de classes.
- `ExemploPolimorfismo.java`: Demonstração de polimorfismo em Java.
- `ExemploEncapsulamento.java`: Uso de encapsulamento para proteger atributos.
